Decentraliazed Storage System

Decentralized storage systems are a type of data storage system that operate on a network of independent nodes, rather than relying on a centralized server or data center. In a decentralized storage system, data is distributed across a network of nodes, which may be located anywhere in the world. Each node in the network stores a portion of the data, and the system as a whole functions as a peer-to-peer network.

IPFS

PFS is a peer-to-peer protocol for sharing and storing files on a distributed network.

It uses content-addressed hyperlinks to allow nodes in the network to locate and retrieve files.

IPFS is designed to be decentralized, meaning there is no single point of failure or control.

It can be used to store and distribute any type of file, including text, images, videos, and application code.

File coin

Filecoin is a decentralized storage network that allows users to store and retrieve data in a peer-to-peer manner, using a cryptocurrency called FIL as an incentive mechanism for miners to provide storage space.

It was founded in 2014 by Juan Benet and is built on top of the InterPlanetary File System (IPFS) protocol.

Filecoin's storage network is designed to be secure, resilient, and censorship-resistant. The network is made up of independent storage providers who compete to provide the best storage and retrieval services to users. Users can use FIL to pay for storage on the network and miners can earn FIL for providing storage and retrieval services.

Decentralization

Decentralization is the concept of distributing power, decision-making authority, and control away from a central authority or organization and dispersing it among a network of individuals or entities. In a decentralized system, there is no single point of control, and instead, decision-making is distributed across the network.
